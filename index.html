Grep

This webservice allows you to search the English dictionary for words matching a particular pattern. This could be useful for solving crosswords, say.

Pattern syntax

The syntax is to use . for wildcard, ^ for starts with and $ for ends with, so ^b..t$ will match four letter words beginning b and ending t.
